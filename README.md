# Physics-Informed Neural Networks for Differential Equations

This repository contains implementations of Physics-Informed Neural Networks (PINNs) for solving ordinary differential equations using PyTorch and JAX.

## Features

- Initial Value Problems (IVPs)
- Boundary Value Problems (BVPs)
- Soft and Hard Constraint Enforcement
- Quadrature Loss Formulation
- Inverse Problems
- Lotka–Volterra System
- FBPINNs
- ELM-FBPINNs

## Tech Stack

- Python
- PyTorch
- JAX
- NumPy
- Google Colab

## Repository Structure

- `1_Forced Damped Oscillator`
- `2_IVPs`
- `3_BVPs`
- `4_Inverse Problems`
- `5_Lotka Volterra`
- `6_Quadrature Loss`
- `7_Hybrid Euler NN`
- `8_Activation Functions`
- `9_FBPINNs and ELM`
